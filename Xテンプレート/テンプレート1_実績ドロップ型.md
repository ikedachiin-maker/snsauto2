# テンプレート1: 実績ドロップ型（数字インパクト → 解説 → 軽CTA）

## 概要
トミーが最も頻繁に使うパターン。冒頭で衝撃的な実績数字をドロップし、読者の注意を一瞬で掴む。その後、背景を説明し、最後に軽いCTAで行動を促す。

## 心理トリガー
- **社会的証明**: 具体的な数字で信頼性を構築
- **権威性**: 実績で「この人の話は聞く価値がある」と思わせる
- **自虐ユーモア**: 数字の後に自虐を入れて親近感を出す

## テンプレート構造

```
[衝撃的な数字] + [短い状況説明]

[補足データ（LTV、期間、リスト数など具体的指標）]
[自虐 or ぶっちゃけコメント（ギャップ演出）]

[なぜこの結果が出たかの簡潔な理由]
[従来の方法との違い（○○なし、△△なし）]

[ターゲットへの問いかけ or 軽いオファー]
[ユーモアで締める]
```

## 実例テンプレート

```
[ツール名/手法名]使ってサクッと[金額]万。

[対象者の説明]から。やはり[ツール名]は[ターゲット層]と相性が良い。

「昔は年商[X]億の[職種]だったのに最近ダメなんだよな…」

という[対象者]、[ツール名]で人生取り戻しませんか？

（このオファー億売れそうw）

[CTA URL]
```

## 応用例

```
AI記事ジェネレーター使ってサクッと月商350万。

元々月5万の副業ブロガーだった方から。やはりAIツールは「コツコツ型」の人と相性が良い。

「毎日3時間かけて記事書いてたのに月5万だった…」

という方、AI記事ジェネレーターで収益10倍にしませんか？

（この手法もっと早く知りたかったw）

[CTA URL]
```

## ポイント
1. 冒頭の数字は「万」単位以上で驚きを与える
2. 自虐やユーモアで「売り込み感」を消す
3. CTAは問いかけ型にして押し付けない
4. 最後にwやユーモアで軽く締める
5. 画像: 実績スクリーンショット or 数字を大きく表示したタイトル画像
