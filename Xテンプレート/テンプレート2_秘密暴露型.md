# テンプレート2: 秘密暴露型（好奇心ギャップ → チラ見せ → エンゲージメントCTA）

## 概要
「9割の人が知らない」「実は...」で始め、秘密を暴露する形式。核心はチラ見せに留め、「知りたい人はいいね」でエンゲージメントを最大化する。トミーの中でも最もバイラル力が高いパターン。

## 心理トリガー
- **好奇心ギャップ**: 「知らない」と言われると知りたくなる
- **希少性**: 「私以外に1社だけ」で独占感
- **逆説**: あえてネガティブ要素を入れて興味を引く
- **参加型CTA**: 「いいねで教えて」で双方向感

## テンプレート構造

```
実は[X]割の人が知らない[分野/手法名]「[キャッチーな名前]」というものがある。

リサーチした限り[使っている人の少なさを強調]。
[競合の実績を比較対象として提示]

この手法は「[あえてネガティブな特徴]」だが[逆説的な結果]。

[具体的なデータ or メリットを1-2個チラ見せ]

この手法を知りたい人いますか？
いたら「いいねポチ」で教えてください。
[次のアクションを予告]

[CTA URL]
```

## 実例テンプレート

```
実は9割の人が知らないSNS集客手法「ゼロローンチ」というものがある。

リサーチした限り日本で使っているのは3社だけ。
（うち1社はこの手法だけで月商2000万を安定的に出している）

この手法は「フォロワー0人でも、広告費0円でも機能する」という常識破りの仕組み。

実際に僕のクライアントは開始3日でリスト200件を獲得した。

この手法の全貌を知りたい人いますか？
いたら「いいねポチ」で教えてください。企画化検討します。

[CTA URL]
```

## ポイント
1. 「X割の人が知らない」は7割〜9割が最適（信憑性のバランス）
2. ネガティブ要素を入れることで信頼性が上がる（「情弱狩り」「教育なし」等）
3. CTAは「いいね」など低ハードルの行動を要求
4. 「企画化検討します」で今後のコンテンツへの期待を作る
5. 画像: 秘密感のあるダーク背景 + 衝撃的なキャッチコピー
