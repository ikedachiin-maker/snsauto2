# テンプレート3: 対比分析型（A vs B → 数字で証明 → 推奨）

## 概要
2つの選択肢を数字で比較し、明確な勝者を提示するパターン。「note vs Park」「手動 vs AI」など、読者が迷っている2択に対して明確な答えを出す。データドリブンで説得力が高い。

## 心理トリガー
- **対比効果**: 2つ並べることで差が際立つ
- **社会的証明**: 具体的な数字で客観性を演出
- **損失回避**: 「間違った選択をしている」と気づかせる
- **権威性**: 自分の実体験データで語る

## テンプレート構造

```
[選択肢A]で[期間]で[金額A]稼いだ。
[選択肢B]で[同期間]で[金額B]稼いだ。

時代は[選択肢B]。

[なぜ差が出るのかの分析（2-3行）]

[多くの人が選択肢Aを使い続けている理由]
[でも実際のデータはこう（対比の深掘り）]

[結論：具体的な推奨アクション]

[CTA URL]
```

## 実例テンプレート

```
メルマガで3ヶ月で売上80万だった。
LINEに切り替えて3ヶ月で売上620万になった。

時代はLINE...と言いたいところだが、実は違う。

メルマガの到達率は15%。LINEの到達率は90%。
でも本質はそこじゃない。

LINEは「即レス文化」があるから、セールスの温度感が全然違う。
メルマガは「後で読もう」→そのまま忘れる。
LINEは「今見た」→衝動で購入。

同じコンテンツ、同じ価格、同じコピーでも
「どこで売るか」で8倍の差が出る。

あなたは今、どこで売っていますか？

[CTA URL]
```

## ポイント
1. 必ず具体的な数字の対比から始める（抽象的な比較NG）
2. 「時代は○○」と一言で結論を出す（明快さが重要）
3. 表面的な比較に留まらず、「本質はそこじゃない」と深掘りする
4. 読者の現状を問いかけて自分ごと化させる
5. 画像: 比較表・グラフ・Before/After型のスライド
