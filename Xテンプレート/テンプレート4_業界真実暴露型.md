# テンプレート4: 業界真実暴露型（業界批評 → 真実の暴露 → ポジション表明）

## 概要
「○○年この業界にいて分かったこと」という形で、業界の裏側やタブーに触れるパターン。長年の経験に基づく権威性と、タブーを語る勇気で信頼を獲得。二極化戦略で熱狂的なファンを作る。

## 心理トリガー
- **権威性**: 「12年」「14年」の業界経験
- **タブー破り**: 業界で誰も言わないことを言う勇気
- **二極化**: 賛否が分かれることであえてポジションを取る
- **インサイダー感**: 「業界の中にいる人だけが知っている」

## テンプレート構造

```
[業界名]を[X]年続けて「[テーマ]」は[結論の要約]。

[箇条書きで2-3個の結論を提示]
(1)[結論1]
(2)[結論2]

共通の特徴は「[共通点1]」「[共通点2]」で、
[一般的な通説]は[反証/nuance]。

[多くの人が犯す間違い]

[自分の見解・ポジション表明]

以下、解説。

[本文（長文スレッド/画像スライドへ誘導）]

[CTA URL]
```

## 実例テンプレート

```
副業コンサル業界を8年見てきて「消えずに稼ぎ続けている人の共通点」が分かった。

(1)仕組みを持っている（労働収入ではない）
(2)1つのプラットフォームに依存していない

共通の特徴は「地味」で「目立たない」こと。
SNSでバズってる人が最も稼いでいるわけではない。

最も危険なのは「フォロワー数=収入」と思い込むこと。
フォロワー10万人で月収30万の人もいれば、
フォロワー3000人で月商1000万の人もいる。

差を生んでいるのは「仕組み」だけ。

以下、具体的に何をすれば仕組みが作れるか解説する。

[CTA URL]
```

## ポイント
1. 「X年」の経験値は必ず入れる（権威性の根拠）
2. 結論を箇条書きで先に出す（読者の時間を尊重）
3. 通説への反論を必ず入れる（差別化）
4. 「以下、解説」で長文/スレッド/記事への誘導
5. 画像: リスト化されたスライド or インフォグラフィック
6. 議論を呼ぶ内容にしてリプライ/引用RTを促す
