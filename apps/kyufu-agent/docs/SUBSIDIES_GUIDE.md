# 給付金・補助金 詳細ガイド

東京都在住者が申請できる主な給付金・補助金の詳細情報です。

---

## 自動申請対応（優先度高）

### 1. 自転車用ヘルメット購入補助金（渋谷区）
- **金額**: 2,000円
- **対象**: 渋谷区在住者
- **申請方法**: 渋谷区公式LINE
- **必要書類**: 身分証明書（マイナンバーカード・運転免許証・在留カード）
- **申請URL**: https://www.city.shibuya.tokyo.jp/kurashi/kotsu/chiiki-jitensha/jitensyayouherumettokounyuhojo.html
- **自動化難易度**: ★☆☆（簡単）
- **備考**: 渋谷区以外でも類似制度あり（千代田区・港区・文京区等）。他の区に住んでいる場合は各区のサイトを確認。

### 2. 018サポート（東京都）
- **金額**: 月額5,000円 × 12ヶ月 = 年間60,000円
- **対象**: 東京都在住の0〜18歳の子供がいる家庭（所得制限なし）
- **申請方法**: Webフォーム
- **必要書類**: 本人確認書類、子供の証明書類
- **申請URL**: https://018support.metro.tokyo.lg.jp/register_ex/
- **自動化難易度**: ★★☆（普通）
- **備考**: 子供1人につき月5,000円。複数の子供がいる場合はその分支給される。

### 3. 赤ちゃんファースト（東京都出産・子育て応援事業）
- **金額**: 最大10万円相当のギフト（出産応援5万円＋子育て応援5万円）
- **対象**: 東京都在住の妊婦・出産後の保護者
- **申請方法**: 専用Webサイト
- **必要書類**: 妊娠届または出生届の提出が前提
- **申請URL**: https://www.fukushi.metro.tokyo.lg.jp/kodomo/shussan/tokyo_shussankosodateouen
- **自動化難易度**: ★★☆（普通）
- **備考**: 現金ではなくギフトカタログ形式。育児用品・子育て支援サービスと交換。

### 4. 児童手当
- **金額**: 月額10,000〜30,000円（年齢・人数により異なる）
  - 3歳未満: 月15,000円
  - 3歳〜小学校修了: 月10,000円（第3子以降は月30,000円）
  - 中学生: 月10,000円
- **対象**: 中学校修了前の子供がいる家庭（所得制限なし）
- **申請方法**: 各区市町村のWebフォームまたは窓口
- **必要書類**: 申請者の健康保険証、通帳、印鑑
- **自動化難易度**: ★★☆（普通）
- **備考**: 出生後15日以内に申請すると遡及支給あり。

### 5. 地方移住支援金
- **金額**: 単身60万円、世帯100万円（子供1人追加ごと+100万円）
- **対象**: 東京23区在住または通勤者が地方に移住する場合
- **申請方法**: 移住先の市区町村Webフォーム
- **必要書類**: 住民票、就業証明書等
- **申請URL**: https://www.soumu.go.jp/main_sosiki/jichi_gyousei/c-gyousei/02gyosei08_03000066.html
- **自動化難易度**: ★★★（難しい）
- **備考**: 移住先での就業が条件。移住後1年以内に申請。

## 手動申請が必要なもの

### 6. 高齢者インフルエンザ予防接種費用助成
- **金額**: 2,000円程度（区市町村により異なる）
- **対象**: 65歳以上（または60〜64歳で一定の障害がある方）
- **申請方法**: 医療機関での接種時に直接助成（事前申請不要）
- **備考**: 医療機関に接種券を持参するだけ。事前のオンライン申請は不要。

### 7. 運転免許自主返納支援
- **金額**: 区市町村により異なる（図書カード・商品券等）
- **対象**: 65歳以上で免許を自主返納した方
- **申請方法**: 警察署で返納後、各区市町村窓口で申請
- **備考**: 荒川区は図書カード5,000円相当など。区によって金額・内容が異なる。

### 8. 高等学校等就学支援金
- **金額**: 年額最大39万6,000円
- **対象**: 高校生がいる世帯（世帯年収910万円未満目安）
- **申請方法**: 学校経由で申請（学校から案内あり）
- **備考**: 国の制度。東京都は独自の上乗せあり（私立高校授業料実質無償化）。

### 9. 再就職支援（教育訓練給付金）
- **金額**: 最大50万円（受講費用の最大70%）
- **対象**: 雇用保険加入歴がある方
- **申請方法**: ハローワークで申請
- **備考**: 専門実践教育訓練給付金は最大70%助成。受講前にハローワークで相談必須。

## 申請のポイント

### 申請を忘れやすいタイミング

| タイミング | 申請すべき給付金 |
|-----------|----------------|
| 子供が生まれた時 | 児童手当、赤ちゃんファースト、018サポート |
| 子供が高校入学時 | 高等学校等就学支援金 |
| 65歳になった時 | 高齢者インフルエンザ予防接種助成 |
| 免許を返納した時 | 運転免許自主返納支援 |
| 自転車ヘルメットを買った時 | ヘルメット購入補助金 |
| 地方移住を決めた時 | 地方移住支援金 |
| 省エネ住宅を建てる時 | 東京ゼロエミ住宅 |

### 申請期限に注意

多くの給付金は申請期限があります。特に以下は期限が短いので注意：
- 児童手当: 出生後15日以内（遅れると遡及期間が限られる）
- 赤ちゃんファースト: 出産後一定期間内
- 地方移住支援金: 移住後1年以内
