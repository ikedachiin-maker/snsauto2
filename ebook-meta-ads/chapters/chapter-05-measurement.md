「広告の管理画面が難しすぎて、何を見ればいいかわからない」

これは多くのオーナーさんが感じることです。
でも、実際に見るべき数字はシンプルです。

この章では、**広告の出し方から効果測定・改善のサイクル**まで、スマホで完結できる方法をお伝えします。

## 5-1｜広告配信の基本設定（スクリーンショットで解説）

### Meta広告の作成手順

**【準備】Metaビジネスマネージャーのセットアップ**

まだ設定していない場合は以下の順番で進めてください。

1. `business.facebook.com` にアクセス
2. 「アカウントを作成」→ ビジネス名・メールを入力
3. Facebookページと連携
4. Instagramアカウントを追加

---

### キャンペーン作成の手順

**Step 1：キャンペーンの目的を選ぶ**

「キャンペーンを作成」→ 目的を選択

✅ 「認知度」→「リーチ」を選択

（コンバージョン・トラフィックは少額では非推奨）

**Step 2：広告セット（ターゲット・予算を設定）**

| 項目 | 設定内容 |
|------|---------|
| 予算 | 1日の予算：500〜1,000円からスタート |
| 掲載期間 | 終了日なし（いつでも停止できる） |
| 場所 | 店舗から半径○km（第3章を参照） |
| 年齢 | 業種に合わせて（第3章を参照） |
| 性別 | 業種に合わせて |
| オーディエンス | Advantage+（推奨）または詳細設定 |

**Step 3：広告（クリエイティブ）を設定**

| 項目 | 設定内容 |
|------|---------|
| 形式 | 画像1枚 または 動画 |
| 画像/動画 | 第4章で作成したもの |
| 広告文 | 3〜5行のシンプルなキャプション |
| 見出し | 15文字以内（任意） |
| 掲載面 | 「自動配置」推奨（Instagram・Facebookに自動配信） |

## 5-2｜広告文（キャプション）の書き方

クリエイティブと同様、広告文も「型」があります。

### 認知広告のキャプション4ステップ

```
① 共感（1行）
「毎朝の眉メイクに時間がかかっていませんか？」

② 提案（1〜2行）
「眉アートメイクで、朝のメイク時間を半分に。」

③ 安心感（1行）
「初回カウンセリング無料・完全個室対応。」

④ 行動を促す（1行）
「プロフィールから詳細をチェック👇」
```

### 全文例（眉サロン）
```
毎朝の眉メイクに30分かけていませんか？

眉アートメイクなら、起きた瞬間から理想の眉。
メイク時間が10分以上短縮できます。

初回カウンセリングは無料。
完全個室・完全予約制で安心してご来店いただけます。

プロフィールから詳細をご確認ください👇
@your_account
```

## 5-3｜Advantage+（AI自動最適化）の使い方

2024年以降、Meta公式が最も推奨しているのが**Advantage+キャンペーン**です。

### Advantage+の3つのメリット
1. ターゲット設定をAIが自動最適化
2. 複数のクリエイティブを自動でテスト
3. 少額予算でも学習が進みやすい

### 設定方法
キャンペーン作成時に「Advantage+キャンペーン予算」をONにするだけ。

地域と年齢だけ設定して、残りはAIに任せる。
最初の2週間は数値が安定しないことがありますが、それは正常です。
AIが学習中のためで、3週間後から徐々に安定してきます。

## 5-4｜見るべき数値はたった3つ

管理画面には数十の指標が並んでいますが、認知広告では以下の3つだけ見れば十分です。

### ① CPM（コスト・パー・ミル）
**1,000回表示されるのにかかったコスト**

| 評価 | 目安 |
|------|------|
| 良好 | 500円以下 |
| 普通 | 500〜1,000円 |
| 要改善 | 1,000円以上 |

CPMが高い場合：クリエイティブを変える、ターゲットを見直す

### ② リーチ数
**何人のユニークユーザーに届いたか**

1日500円の予算で、地域ターゲット（半径5km）であれば：
- 都市部：500〜1,500人/日
- 郊外：300〜1,000人/日

が目安です。この数値を下回る場合はオーディエンスが狭すぎる可能性があります。

### ③ フリークエンシー（頻度）
**同一人物に何回表示されたか**

| 評価 | 目安 |
|------|------|
| 良好 | 1.0〜3.0回 |
| 注意 | 3.0〜5.0回 |
| 要停止/変更 | 5.0回以上 |

フリークエンシーが5回を超えると「広告疲れ」が起きます。
この場合はクリエイティブを差し替えるか、オーディエンスを広げる必要があります。

## 5-5｜週次PDCAの回し方

広告は「出して終わり」ではありません。
ただし、毎日確認する必要もありません。

### 推奨：週1回・15分のチェック

**毎週月曜日（推奨）に以下を確認**

```
□ CPMは1,000円以下か？
□ リーチ数は想定通りか？
□ フリークエンシーは5回以内か？
□ Instagramのフォロワーは増えているか？
□ 問い合わせ・予約に変化はあるか？
```

### 改善のアクション早見表

| 数値の状態 | 取るべきアクション |
|-----------|-----------------|
| CPMが高い（1,000円超） | クリエイティブを変更 |
| リーチが少ない | 地域範囲を広げる・予算を増やす |
| フリークエンシーが高い | クリエイティブを差し替え |
| フォロワーが増えない | Instagramプロフィールを見直す |
| 全体的に低調 | 2週間待つ（AI学習中の可能性） |

### クリエイティブのローテーション
同じクリエイティブを1ヶ月以上使い続けると効果が落ちます。
**月1回**は新しいクリエイティブを追加しましょう。

目安：常に2〜3種類のクリエイティブを並走させる

## 5-6｜広告とInstagramのつながりを確認する方法

認知広告の最終的な成果は「Instagramフォロワーの増加」として現れます。

### 確認すべきInstagramのインサイト

Instagramビジネスアカウント → プロフィールページ → 「インサイト」

**見るべき数値**
- プロフィールへのアクセス数（週次）
- フォロワー増加数（週次）
- リーチ数の推移

広告を出した週とそうでない週を比較することで、広告の効果を間接的に測定できます。

- 広告の目的は「認知度アップ（リーチ）」を選ぶ
- Advantage+を使えば、ターゲット設定はAIに任せられる
