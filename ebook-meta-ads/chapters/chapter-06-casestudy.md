# 第6章｜実践ケーススタディ

## 実際に少額予算で成果を出したサロンの事例

---

「本当に少額で成果が出るのか？」

百聞は一見にしかず。この章では、実際のサロン運営者が少額のMeta認知広告とInstagram運用を組み合わせて成果を出した事例を業種別にご紹介します。

※事例はプライバシー保護のため、一部情報を変更しています。

## 6-1｜エステサロン事例：1日800円・月3名の新規集客

### サロン概要
- 場所：神奈川県・住宅街エリア
- 業態：個人経営のフェイシャルエステ（1人オーナー）
- 開始前の状況：Instagram投稿は週1〜2回、フォロワー約200名、広告経験ゼロ

### 実施した内容

**Instagramの整備（広告開始前の2週間）**
- プロフィール文を業種特化・地域名入りに変更
- ハイライト（メニュー・料金・施術の流れ・よくある質問）を整備
- Before/After投稿を6枚まとめてアップ

**Meta認知広告の設定**
| 項目 | 内容 |
|------|------|
| 月額予算 | 約25,000円（1日800円） |
| 目的 | リーチ（認知度アップ） |
| 地域 | 店舗から半径6km |
| 年齢・性別 | 30〜50歳・女性 |
| オーディエンス | Advantage+ |
| クリエイティブ | Before/After画像2種類 |

### 3ヶ月後の結果

| 指標 | 開始前 | 3ヶ月後 |
|------|--------|---------|
| Instagramフォロワー | 200名 | 480名 |
| プロフィールへのアクセス（月） | 約80回 | 約420回 |
| 新規問い合わせ（月） | 1〜2名 | 4〜5名 |
| CPM（平均） | — | 約620円 |

### オーナーの感想
> 「広告を出す前にInstagramを整えることが大切だと実感しました。最初の1ヶ月はフォロワーが少しずつ増えるだけでしたが、2ヶ月目から問い合わせが来るようになりました。月2万5千円の投資で新規客が3〜4名増えたので、費用対効果は十分だと思っています。」

## 6-2｜パーソナルジム事例：体験申込みCPA 2,300円

### サロン概要
- 場所：大阪府・駅近商業エリア
- 業態：女性専用パーソナルジム（トレーナー1名）
- 開始前の状況：Instagram投稿なし（アカウントのみ存在）、新規集客は紹介のみ

### 実施した内容

**Instagramの整備（広告開始前の3週間）**
- プロフィール：「女性専用・30代からのボディメイク専門」に特化
- 投稿12本を先にまとめてアップ（ビフォーアフター・食事指導・トレーニング動画）
- LINEへの誘導リンクを設置（体験申込みフォーム）

**Meta認知広告の設定**
| 項目 | 内容 |
|------|------|
| 月額予算 | 約30,000円（1日1,000円） |
| 目的 | リーチ（認知度アップ） |
| 地域 | 店舗から半径5km |
| 年齢・性別 | 28〜45歳・女性 |
| オーディエンス | 興味関心：ダイエット・フィットネス・ボディメイク |
| クリエイティブ | リール動画（トレーナーが語りかける形式）1本 |

### 2ヶ月後の結果

| 指標 | 開始前 | 2ヶ月後 |
|------|--------|---------|
| Instagramフォロワー | 0名 | 230名 |
| 体験申込み（月） | 0〜1名（紹介のみ） | 月5〜6名 |
| 体験申込みCPA | — | 約2,300円 |
| CPM（平均） | — | 約780円 |

### ポイント
体験申込みCPA（顧客獲得単価）が2,300円というのは、パーソナルジム業界では非常に低い数値です。一般的な代理店経由のコンバージョン広告では5,000〜20,000円かかることも珍しくありません。

認知広告→Instagram→LINE誘導という流れで「信頼を積み上げてから申込み」という経路が、結果的に低CPAを実現しました。

## 6-3｜眉サロン事例：Instagramフォロワー+200名/月

### サロン概要
- 場所：東京都・住宅街（最寄り駅から徒歩8分）
- 業態：眉スタイリング専門サロン（オーナー1名）
- 開始前の状況：Instagram活用なし、集客はホットペッパーのみ

### 実施した内容

**Instagramの整備**
- Before/Afterを施術ごとに必ず投稿（週3〜4本）
- 「あなたに似合う眉の形はこれ」シリーズで保存率の高い投稿を量産
- ハイライト：眉の種類・料金・アクセス・FAQ

**Meta認知広告の設定**
| 項目 | 内容 |
|------|------|
| 月額予算 | 約15,000円（1日500円） |
| 目的 | リーチ（認知度アップ） |
| 地域 | 店舗から半径4km |
| 年齢・性別 | 22〜42歳・女性 |
| オーディエンス | Advantage+ |
| クリエイティブ | Before/After画像（月2種類ローテーション） |

### 3ヶ月後の結果

| 指標 | 開始前 | 3ヶ月後 |
|------|--------|---------|
| Instagramフォロワー | 85名 | 720名（+635名） |
| フォロワー増加数（月平均） | 約5名/月 | 約200名/月 |
| ホットペッパー依存度 | 売上の90% | 売上の60%（Instagram経由が増加） |
| CPM（平均） | — | 約480円 |

### オーナーの感想
> 「月1万5千円でこれだけフォロワーが増えるとは思っていませんでした。ホットペッパーの手数料が高くて悩んでいたのですが、少しずつInstagram経由の予約が増えてきて、手数料の負担が減っています。」

## 6-4｜脱毛サロン事例：リール広告でカウンセリング予約率が向上

### サロン概要
- 場所：愛知県・郊外ロードサイド
- 業態：女性脱毛専門サロン（スタッフ2名）
- 開始前の状況：Meta広告経験あり（コンバージョン広告）→ CPAが高すぎて停止中

### 転換点：コンバージョン広告から認知広告へ

このサロンは以前、コンバージョン広告（予約獲得目的）を試みましたが、月30,000円の予算ではデータが集まらず、CPA（顧客獲得単価）が1件15,000円を超えていました。

そこで戦略を変更し、**認知広告＋Instagram強化**に切り替えました。

**Meta認知広告の設定（変更後）**
| 項目 | 内容 |
|------|------|
| 月額予算 | 約30,000円（1日1,000円） |
| 目的 | リーチ（認知度アップ）に変更 |
| 地域 | 店舗から半径8km（ロードサイドのため広めに設定） |
| 年齢・性別 | 20〜38歳・女性 |
| クリエイティブ | UGC風リール動画（スタッフが語りかける形式） |

**Instagram強化の施策**
- 「脱毛のよくある疑問Q&A」シリーズを週2本投稿
- 「初めての脱毛、何から始めればいい？」という教育コンテンツ
- ストーリーズで「無料カウンセリングのご案内」を毎週1回

### 3ヶ月後の結果

| 指標 | 変更前（コンバージョン広告） | 変更後（認知広告） |
|------|--------------------------|-----------------|
| 月額広告費 | 30,000円 | 30,000円（同額） |
| 月間カウンセリング予約 | 2〜3件 | 6〜8件 |
| CPA（顧客獲得単価） | 約15,000円 | 約4,500円 |
| Instagramフォロワー | 120名 | 580名 |

### なぜ認知広告の方がCPAが下がったのか

コンバージョン広告は「今すぐ買う人」にしか届きません。
少額予算では、そもそも「今すぐ買う人」が少なすぎてデータが集まらない。

一方、認知広告は「まだ知らなかった人・検討段階の人」に広く届きます。
Instagramで信頼を積み上げることで、検討段階の人が予約に至るまでの時間は長くなりますが、最終的なCPAは大幅に下がりました。
