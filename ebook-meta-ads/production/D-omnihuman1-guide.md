## Step 1：音声収録

### 収録環境の整え方
```
【必須】
□ 静かな部屋（エアコンの音もNG）
□ マイク付きイヤホン（スマホ付属品でOK）
□ スマホを口から20〜30cm離して固定

【あると良い】
□ 毛布や布団で囲む（反響音を防ぐ）
□ 夜間収録（外部騒音が少ない）
```

### 収録手順

**Step 1-1：台本を印刷または大きな画面で表示**
`step5-vsl-script.md` の台本を準備。

**Step 1-2：試し読み2〜3回**
本番前に必ず通し読みをする。
目安タイム：15〜17分に収まるか確認。

**Step 1-3：本番収録**
```
スマホの「ボイスメモ」アプリで録音
→ 形式：M4A（後でMP3変換）
→ 目安：16〜18分（余裕を持って）

【収録のコツ】
- ゆっくり、はっきり話す（普段の1.2倍ゆっくり）
- 「。」で1秒、「、」で0.5秒の間を置く
- 噛んだ箇所は止まらず最後まで読み切る
  （後でカット編集できる）
- 笑顔で話す（声のトーンが変わる）
```

**Step 1-4：音声ファイルをMP3に変換**
```
iPhoneの場合：
「ファイル」アプリ → ボイスメモを「共有」
→ 「GarageBand」で開いてMP3書き出し

Androidの場合：
録音アプリの設定でMP3形式を選択
または「Audio Converter」アプリを使用

ファイル名：vsl-main.mp3
```

## Step 2：アバター画像の準備

### 選択肢A：自分の写真を使う（推奨）
```
【写真の条件】
□ 正面を向いている（顔が中心）
□ 表情が自然（固すぎない笑顔）
□ 背景がシンプル（白壁・無地）
□ 解像度：最低1080×1080px以上
□ 形式：JPG または PNG

【撮影のコツ】
- スマホを目の高さに固定（三脚推奨）
- 自然光（窓際）が最も自然な仕上がり
- ビジネスカジュアルな服装
- リングライトがあれば使用（2,000〜3,000円）
```

### 選択肢B：OmniHuman1のプリセットアバターを使う
```
OmniHuman1サイト内のアバターギャラリーから選択
→「30代女性・ビジネスカジュアル・親しみやすい」系を選ぶ
→ 著作権フリーで商用利用可能
```

### 選択肢C：AIで理想のアバター画像を生成する
NanoBanana Pro（Google AI Studio）で以下のプロンプトを使用：

```
Create a professional headshot photo for video avatar use.

Subject: Japanese woman in her early 30s
Expression: Warm, genuine smile, approachable and trustworthy
Eye contact: Looking directly at camera

Appearance:
- Smart casual outfit (light blue or white blouse)
- Natural makeup, clean professional look

Background: Clean white or very light gray (#F5F5F5)
Lighting: Soft, even, flattering (studio-style)

High resolution, sharp focus on face

Size: 1080x1080px
```

## Step 3：OmniHuman1でリップシンク動画生成

### アクセス・ログイン
```
1. omnihuman1.org/ja にアクセス
2. アカウント登録（メールアドレスで無料登録可能）
3. ダッシュボードへ移動
```

### 動画生成手順
```
【Step 3-1】「新規プロジェクト」をクリック

【Step 3-2】アバター画像をアップロード
→ Step 2で準備した画像（JPG/PNG）

【Step 3-3】音声ファイルをアップロード
→ vsl-main.mp3

【Step 3-4】設定を確認
・解像度：1920×1080（Full HD）
・フレームレート：30fps
・背景：そのまま or バーチャル背景を選択

【Step 3-5】「生成開始」をクリック
→ 15分の動画で約15〜30分かかる
→ 完了通知メールが届く

【Step 3-6】プレビューで確認
→ 口の動きが音声と合っているか確認
→ 不自然な箇所があれば再生成

【Step 3-7】MP4でダウンロード
→ ファイル名：vsl-avatar-raw.mp4
```

## Step 4：CapCutで後処理・完成

### 追加する要素
```
① テロップ（重要ポイントを文字で強調）
② BGM（控えめなBGM・15%音量）
③ オープニング画面（5秒）
④ エンディング画面（5秒）
⑤ CTA画面（申込みURL表示・10秒）
```

### CapCut編集手順

**4-1：プロジェクト作成**
```
CapCutを開く → 「新規プロジェクト」
→ vsl-avatar-raw.mp4 を読み込む
→ 縦横比：16:9（横型）を選択
```

**4-2：テロップ追加（重要箇所のみ・全体の20%程度）**
```
【追加推奨タイミング】
00:00〜00:45「1日500円から始めるMeta広告集客術」
03:00〜03:10「3つの壁」
06:00〜06:20「ステップ1・2・3」
09:30〜09:45「個別コンサルの内容」
13:00〜13:10「Kindle読者限定価格」
14:30〜15:00「申込みはこちら👇[URL]」

【テロップのスタイル】
・白文字・黒縁取り・中央下部に配置
・フォント：太め（CapCutの「太丸ゴシック」系）
・サイズ：画面幅の60〜70%
```

**4-3：BGM追加**
```
「音声」→「BGMを追加」→「おすすめ」から選択
→ 「Soft Piano」「Acoustic」「Calm」系を選ぶ
→ 音量：10〜15%（声が主役）
→ 動画全体にフェードイン・フェードアウト設定
```

**4-4：オープニング画面（冒頭5秒に追加）**
```
黒背景 + 白テキスト：
「エステ・ジム・眉サロン・脱毛サロン
 オーナーの方へ」

→ フェードイン効果
```

**4-5：CTA画面（最後10秒に追加）**
```
背景：紺色（#0F172A）

テキスト1：「個別コンサルの詳細・お申し込み」（白・大）
テキスト2：「↓ 下のURLをタップ ↓」（黄色）
テキスト3：「[申込みページURL]」（白・下線付き）

→ フェードイン効果
```

**4-6：書き出し設定**
```
「エクスポート」をタップ
→ 解像度：1080p
→ フレームレート：30fps
→ ファイル名：vsl-final.mp4
→ ファイルサイズ確認（100MB以下推奨）
```

## Step 5：YouTube/Vimeoにアップロード

### YouTube限定公開（推奨）
```
【設定】
・公開設定：「限定公開」（URLを知っている人のみ視聴可）
・タイトル：設定しない（または内部管理用の名前）
・サムネイル：VSL用に別途作成（または自動生成）
・コメント：オフ
・年齢制限：なし

【アップロード後】
→ 動画URLをコピー
→ UTAGEの「Day 3メッセージ」の [VSL動画URL] に貼り付け
```

## よくあるトラブル

| 問題 | 解決策 |
|------|--------|
| 口の動きがズレる | 音声ファイルをWAV形式で再試行 |
| 画質が荒い | アバター画像を高解像度に変更（2000px以上） |
| CapCutで動作が重い | 画質を720pに下げて編集、書き出し時に1080pに |
| ファイルサイズが大きい | CapCutの「スマートHDR」をオフにする |
| 審査で限定公開が弾かれる | Vimeoのプライベート設定に切り替え |
