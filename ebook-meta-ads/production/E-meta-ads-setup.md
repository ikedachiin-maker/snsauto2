# E｜Meta認知広告 入稿・配信設定ガイド
## Kindle集客＋Instagramフォロワー獲得用

---

## キャンペーン全体設計

```
目的：認知度アップ（リーチ）
予算：1日500〜1,000円からスタート
地域：店舗から半径5km
期間：終了日なし（常時運用）
クリエイティブ：Before/After画像 + リール動画を並走
```

---

## 事前準備チェックリスト

```
□ Facebookページ作成済み（ビジネス名で）
□ Instagramビジネスアカウント作成済み
□ Facebookページ ↔ Instagram連携済み
□ Metaビジネスマネージャー作成済み
   （business.facebook.com）
□ 広告アカウント作成済み（act_XXXXXXXXX）
□ お支払い方法登録済み（クレジットカード）
□ クリエイティブ素材準備済み
   （chapter-04-creative.md を参照）
```

## Step 1：Metaビジネスマネージャーのセットアップ

### 1-1：アカウント作成
```
① business.facebook.com にアクセス
② 「アカウントを作成」をクリック
③ ビジネス名・氏名・メールアドレスを入力
④ Facebookページを追加
⑤ Instagramアカウントを追加
   （設定 → アカウント → Instagramアカウント → 追加）
⑥ 広告アカウントを追加または新規作成
```

### 1-2：Facebookピクセルの設置（推奨）
```
ビジネスマネージャー → 「イベントマネージャー」
→ 「データソースを追加」→「ウェブ」→「ピクセル」
→ 予約ページのURLを入力
→ コードを予約ページのheadタグに設置

※ ピクセルがなくても認知広告は出稿可能。
  将来のコンバージョン広告を見据えて設置推奨。
```

---

## Step 2：キャンペーン作成

### 2-1：新規キャンペーン
```
広告マネージャー → 「作成」をクリック

【目的の選択】
「認知度」→「リーチ」を選択

【キャンペーン名】
「認知_Kindle集客_[業種名]_[開始日]」
例：「認知_Kindle集客_エステ_20260401」
```

### 2-2：Advantage+キャンペーン予算
```
「Advantage+キャンペーン予算」：ON（推奨）

【1日の予算設定】
スタート時：500〜1,000円/日
3ヶ月後に効果確認：1,000〜2,000円/日に増額

※ Advantage+をONにすることで
  MetaのAIが複数広告セット間で
  予算を自動最適化する
```

## Step 4：広告（クリエイティブ）設定

### 4-1：広告名
```
「[クリエイティブ種類]_[日付]」
例：「BA画像_眉Before/After_20260401」
   「リール動画_エステ施術中_20260401」
```

### 4-2：広告形式
```
【推奨①】画像（Before/After）
形式：シングル画像
サイズ：1080×1080px（正方形）または1080×1350px（縦型）

【推奨②】動画（リール）
形式：シングル動画
サイズ：1080×1920px（縦型・9:16）
長さ：15〜30秒
```

### 4-3：広告テキスト設定

**エステ向け（コピー例）**

```
初回カウンセリング無料・完全個室。

プロフィールから詳細をご確認ください👇

【見出し（任意・15文字以内）】
「肌が変わる体験を」

【説明文（任意）】
「完全予約制・初回無料カウンセリング」

【行動を促すボタン】
「詳細を見る」
```

## Step 5：確認・入稿

### 入稿前チェックリスト
```
□ キャンペーン目的：リーチ（認知度）
□ 1日の予算：500〜1,000円
□ 地域：店舗から半径○km
□ 年齢・性別：業種に合っている
□ クリエイティブ：画像サイズ規定内
□ 広告テキスト：断定的な健康効果の表現なし
□ リンク先：Instagramプロフィール
□ お支払い方法：有効なクレジットカード登録済み
```

### 審査期間
```
通常：数時間〜24時間以内
美容・健康系：最長48時間かかる場合あり

審査中は「審査中」ステータスで表示される。
「有効」になれば配信開始。
```

## Step 6：週次モニタリング設定

### 確認する数値（週1回・月曜日推奨）

| 指標 | 確認場所 | 良好 | 要改善 |
|------|---------|------|--------|
| CPM | 広告マネージャー > 列カスタマイズ | 〜1,000円 | 1,000円超 |
| リーチ数/日 | 広告マネージャー > リーチ | 想定通り | 極端に少ない |
| フリークエンシー | 広告マネージャー > 列カスタマイズ | 1〜3回 | 5回超 |
| Instagramフォロワー増加 | Instagramインサイト | 週5〜20名増 | 変化なし |

### 改善アクション早見表

```
CPMが1,000円超      → クリエイティブを差し替え
リーチが少ない       → 地域半径を1〜2km広げる
フリークエンシー5超  → 新しいクリエイティブを追加
フォロワーが増えない  → Instagramプロフィールを見直す
全体的に低調が2週間  → そのまま待つ（AI学習中）
```

## Step 7：クリエイティブローテーション計画

### 月次ローテーション（推奨）

```
【月1回の作業】
□ 新しいBefore/After画像を1枚追加
□ 新しいリール動画を1本追加
□ パフォーマンスが低いクリエイティブを停止
□ 常に2〜3種類を並走させる

【季節対応（脱毛・エステ）】
3月：「春の肌ケア」「夏に向けて」系クリエイティブ
12月：「年末年始・新生活」系クリエイティブ
→ 1〜2ヶ月前から差し替え準備
```

## A/Bテスト設計（3ヶ月後〜）

3ヶ月の基礎データが集まったら以下をテスト：

```
【テスト①：クリエイティブ比較】
A：Before/After画像
B：リール動画（施術中）
→ CPMが低い方を主軸に

【テスト②：キャプション比較】
A：問いかけ型「○○で悩んでいませんか？」
B：数字型「月3名の新規客が来るようになった話」
→ プロフィールへのアクセス数で判断

【テスト③：地域半径比較】
A：半径3km
B：半径6km
→ CPMとフォロワー増加率で判断
```

## よくあるエラーと対処法

| エラー | 原因 | 対処 |
|-------|------|------|
| 広告が審査で却下 | 表現・画像がポリシー違反 | 第4章のNG表現を確認して修正 |
| 配信量が少ない | オーディエンスが狭すぎる | 地域半径を広げる |
| アカウントが制限 | 支払い失敗・ポリシー違反 | カード情報確認・審査結果を確認 |
| CPMが急上昇 | 競合増加・季節要因 | クリエイティブ差し替えで様子見 |
| フリークエンシー急上昇 | オーディエンスが小さすぎる | 地域を広げる or 新クリエイティブ投入 |
