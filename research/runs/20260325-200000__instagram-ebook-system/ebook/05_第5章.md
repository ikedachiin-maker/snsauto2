## 5-1: なぜクリエイティブが「認知」を左右するのか

認知広告においてクリエイティブとは、単なる「画像や動画」ではありません。ブランドを初めて知るユーザーに「このブランドは何者か」「信頼できるか」「自分に関係があるか」を瞬時に伝えるツールです。

特にインスタグラムでは、ユーザーが1枚の投稿を見る時間は平均1〜3秒と言われています。この短い時間の中で印象を残せるかどうかが、広告の成否を決めます。

**クリエイティブが認知広告に与える影響:**

| 要素 | 効果（研究データより） |
|------|------------------|
| 動画冒頭3秒以内にブランド表示 | 広告想起率+131% |
| ロゴを左上に配置 | 広告想起率+131% |
| ブランド名を見出しに記載 | CTR（クリック率）+28% |
| 製品デモ型広告（実際の使用シーン） | 抽象的なメッセージ広告比で1.5倍のブランド認知 |

これらのデータはMeta社が公表している研究結果をもとにしています。一つひとつは小さな工夫ですが、組み合わせることで大きな差になります。

## 5-2: フォーマット別クリエイティブの基本

インスタグラムには主に3つの広告フォーマットがあります。それぞれ特性が異なるため、目的に合わせて使い分けましょう。

### ① フィード広告（正方形・縦型）

**特徴:**
- タイムラインに自然に溶け込む
- 静止画・動画どちらにも対応
- じっくり見てもらえる傾向

**推奨スペック:**
- サイズ: 1080×1080px（1:1）または1080×1350px（4:5縦型）
- 動画の長さ: 最大60秒（認知目的では15〜30秒推奨）
- ファイル形式: JPG/PNG（静止画）、MP4（動画）

**CPM参考値（グローバル）**: 約$7.68（約1,150円）
**CTR**: 約1.01%

フィード広告はCPMが高め（他フォーマット比）ですが、視認性が高く、じっくり内容を届けたい場合に向いています。

### ② ストーリーズ広告（縦型フルスクリーン）

**特徴:**
- 全画面表示で没入感が高い
- ユーザーが次のストーリーズへタップする間に表示される
- 15秒以内の短い動画が効果的

**推奨スペック:**
- サイズ: 1080×1920px（9:16縦型）必須
- 動画の長さ: 最大60秒（ただし15秒以内を強く推奨）
- 上下150px（セーフゾーン）にテキストや重要な要素を置かない

**CPM参考値（グローバル）**: 約$6.25（約940円）
**CTR**: 約1.34%

ストーリーズはCTRが最も高いフォーマットです。スワイプアップ（現在は「リンクステッカー」に変更）でサイトへ誘導しやすく、行動喚起型の広告に向いています。

### ③ リール広告（縦型短尺動画）

**特徴:**
- インスタグラムで最も視聴されているフォーマット
- TikTok風のエンタメ性の高いコンテンツが好まれる
- アルゴリズムによる拡散効果がある
- 有機投稿との境界が曖昧で広告感が薄い

**推奨スペック:**
- サイズ: 1080×1920px（9:16縦型）必須
- 動画の長さ: 3〜60秒（15〜30秒が最適）
- 音声あり前提で制作する（80%以上のユーザーが音ありで視聴）

**CPM参考値（グローバル）**: 約$5〜$6（約750〜900円）
**CTR**: 約0.76%

リールはCPMが最も安く、コストパフォーマンスに優れています。認知広告として多くの人に届けたい場合、リールを中心に据えるのがおすすめです。

## 5-3: 冒頭3秒の設計法（+131%の広告想起率）

動画広告で最も重要なのが「冒頭3秒」です。ユーザーが動画を途中でスクロールしてしまう前に、ブランド名・商品名を記憶に刻む必要があります。

**研究データが示す事実:**

> 動画の冒頭3秒以内にブランドを明示した広告は、そうでない広告と比べて**広告想起率が131%高い**

これは「ブランド名を最後に出すと印象的」という従来の映像演出の常識とは逆です。広告は映画ではありません。スキップされる前提で設計する必要があります。

### 冒頭3秒の設計テンプレート

**パターン1: ブランド名直球型**

```
[0〜0.5秒] ブランドロゴ（左上に固定表示）
[0〜3秒]   「〇〇（ブランド名）がお届けする〜」というナレーション
[3〜10秒]  商品の魅力・解決策の提示
[10〜15秒] CTAと再度ブランド名
```

**パターン2: 問題提起型**

```
[0〜0.5秒] ブランドロゴ（左上に固定表示）
[0〜3秒]   「〇〇で悩んでいませんか？」という問いかけ（テキスト表示）
[3〜10秒]  商品による解決の実演（デモ）
[10〜15秒] ブランド名＋CTA
```

**パターン3: UGC（ユーザー生成コンテンツ）風型**

```
[0〜0.5秒] ブランドロゴ（左上に固定表示）
[0〜3秒]   「正直に言います、〇〇を使ってみて…」という話しかけ
[3〜10秒]  実際の使用感・変化を語る
[10〜15秒] ブランドURL/アカウント名
```

### ロゴの配置ルール

**「左上固定」が131%広告想起率向上の鍵:**

- ロゴは動画の左上（視線が最初に向かう場所）に配置する
- 透明度は高くせず、はっきりと見える状態にする（白または黒で縁取りがあると映像に関わらず視認性が上がる）
- 動画全体を通して表示し続ける（フェードアウトしない）

**Canvaでのロゴ配置手順（後述のCanva AIセクションで詳しく説明）:**

1. デザイン画面の左上にロゴ画像を配置します
2. テキスト枠の場合は「固定」機能でアニメーションを「なし」に設定します
3. 動画出力後、テスト再生でロゴが常時表示されているか確認します

## 5-4: CanvaAIを使ったクリエイティブ制作

Canva（キャンバ）は、デザインの専門知識がなくてもプロ品質のビジュアルを作れる無料ツールです。スマートフォンアプリ版があり、いつでもどこでも制作できます。2024年以降はAI機能が大幅に強化され、テキストから画像を生成したり、背景を自動で変えたりすることができます。

### Canva のインストールと初期設定

1. App Store（iOS）または Google Play（Android）で「Canva」を検索してインストールします
2. 無料アカウントを作成します（Googleアカウントでのログイン可）
3. 「Canva Pro」への誘導が出ますが、最初は無料版で十分です（AI機能の一部はPro限定）

### インスタグラム広告用テンプレートの選び方

1. Canvaを開いて「+ 作成」をタップします
2. 検索欄に「インスタグラムリール」または「インスタグラムストーリー」と入力します
3. 縦型（9:16）のテンプレートが表示されます
4. 自社ブランドの雰囲気（シンプル・カラフル・高級感など）に合ったテンプレートを選びます

> **無料vs有料テンプレート**: 王冠マーク（♛）のテンプレートはPro限定です。無料テンプレートでも十分おしゃれなものが多数あります。

### CanvaAI機能を使った画像生成

**「テキストを画像に変換（Text to Image）」の使い方:**

1. デザイン画面で「要素」→「AI画像ジェネレーター」を選択します
2. 生成したい画像の説明を日本語または英語で入力します
3. スタイル（写真風・イラスト風・水彩など）を選択します
4. 「画像を生成」をタップします
5. 4パターンが生成されるので、最適なものを選択します

**効果的なプロンプト例（スキンケアブランド）:**

```
Japanese woman in her 30s applying skincare cream,
natural lighting, soft background, professional photography style,
lifestyle image
```

```
Minimalist skincare products flat lay, white background,
natural lighting, high-end cosmetics, clean aesthetic
```

### Canvaで動画を編集する手順

**スマートフォンで撮影した動画を使う場合:**

1. デザイン画面の「動画」タブから「アップロード」をタップします
2. スマートフォンのカメラロールから動画を選択します
3. タイムライン上で動画をトリムします（不要な部分をカット）
4. テキストアニメーションを追加します

**テキストアニメーションの追加（冒頭3秒のブランド表示）:**

1. 「テキスト」→「テキストを追加」をタップします
2. ブランド名を入力します
3. 「アニメーション」→「フェードイン」を選択します
4. テキストの表示タイミングを動画の0秒〜全体表示に設定します

**ロゴの追加と固定:**

1. 「アップロード」からロゴ画像（PNG透過推奨）をアップロードします
2. 動画の左上に配置します
3. 全ページ（全フレーム）に表示させるため「全ページに追加」を選択します

### Canva で完成した動画をエクスポートする

1. 右上の「共有」ボタンをタップします
2. 「ダウンロード」→「MP4動画」を選択します
3. 「ダウンロード」をタップします
4. カメラロールに保存されます

## 5-5: 縦型9:16の基本ルール

リール・ストーリーズ広告は必ず縦型（9:16）で制作します。横型や正方形のクリエイティブを無理やり縦型に変換すると、余白が多くなり視認性が落ちます。

### 縦型9:16のレイアウト設計

**セーフゾーンを意識した配置:**

```
[上部 150px] ← ユーザー名・時間表示と重なるため重要情報を置かない
[中央部]     ← メインビジュアルを配置（全体の70%）
[下部 150px] ← CTAボタン・スワイプ表示と重なるため文字を置かない
```

縦型動画の画面構成（1080×1920pxを例にすると）:

| エリア | Y座標 | 用途 |
|-------|-------|------|
| セーフゾーン外（上） | 0〜150px | インターフェース表示エリア |
| 安全エリア（上） | 150〜400px | ロゴ・ブランド名 |
| メインビジュアル | 400〜1500px | 商品・人物・メインコンテンツ |
| テキストエリア | 1500〜1750px | キャッチコピー・価格・CTA |
| セーフゾーン外（下） | 1750〜1920px | インターフェース表示エリア |

### スマートフォンでの縦型撮影のコツ

1. **カメラを縦向きで持つ**: 当たり前ですが徹底しましょう。横向きで撮ると後から使えなくなります
2. **グリッド線を表示する**: カメラアプリの設定で「グリッド」をオンにすると水平・垂直が保ちやすくなります
3. **被写体は中央〜やや上に配置**: 下部にCTAを入れるスペースを確保するため
4. **明るい場所で撮影する**: 室内なら窓際の自然光が最もきれいに撮れます
5. **手ブレを防ぐ**: 両手でしっかりスマホを持つか、三脚スタンドを使います

## 5-6: 製品デモ型広告が1.5倍効果的な理由

Metaの研究データによると、**製品デモ型広告**（実際に製品を使っている場面を見せる広告）は、抽象的なメッセージ広告（イメージ映像や言葉だけの広告）と比べて**1.5倍のブランド認知（aided brand awareness）**を実現します。

### 製品デモ型広告とは

ユーザーが「この商品は何をするものか・使うとどうなるか」を具体的に理解できる広告です。

**製品デモ型広告の例（スキンケアブランド）:**

- 美容液を手の甲に垂らして伸ばすシーン
- クリームを顔に塗った前後の肌の変化
- 化粧水を手のひらに取り、パッティングしながら吸収させるシーン

**抽象的メッセージ広告の例（効果が低い）:**

- きれいな風景の中でモデルが微笑む映像
- 「美しさへの挑戦」というキャッチコピーのみ
- ブランドロゴが大きく表示されるだけの映像

### 製品デモ動画の撮影手順（スマートフォン版）

**必要なもの:**

- スマートフォン（カメラ性能は現在のスマホなら十分）
- 自社商品
- 白または無地のテーブルクロス・ポスター（背景用）
- 窓際の自然光（または卓上ライト）

**撮影の流れ:**

1. 背景をシンプルに整えます（白い壁・無地の布などが最適）
2. 商品を画面に対して縦向き（9:16）に配置します
3. 商品に焦点を当て、画面の中心〜上部に配置します
4. 商品を手で持ち、実際に使用する動作を3〜10秒撮影します
5. 必要に応じて、使用前・使用中・使用後のシーンを撮影します

## 5-7: UGC風クリエイティブのコツ

UGC（User Generated Content）とは、ユーザーが自発的に投稿したコンテンツのことです。本物のUGCだけでなく、**UGCのような雰囲気で作られた広告**（UGC風広告）も高い効果を発揮します。

### なぜUGC風広告が効くのか

- **広告感が薄い**: 通常の友人投稿のように見えるため、スクロールの手が止まりやすい
- **信頼性が高い**: 一般ユーザーが紹介しているように見えるため、第三者的な安心感がある
- **共感を呼びやすい**: 「自分も同じ悩みがある」という共感が生まれやすい

### UGC風クリエイティブの作り方

**基本スタイル:**

1. **スマホで縦型自撮り動画を撮る**（プロ風の映像ではなく、日常っぽさを出す）
2. **話しかけるトーンで語る**（「えっと」「なんか」などの自然な言葉遣いもOK）
3. **テロップを手書き風フォントにする**（Canvaの「手書き風」フォントを使用）
4. **過度に編集しない**（ロゴやブランドカラーは最小限）

**UGC風動画の台本テンプレート（15秒版）:**

```
[0〜3秒]   「最近ずっと〇〇に悩んでたんですけど、」（自撮り・話しかける）
[3〜8秒]   「これ使ってみたら全然違くて。」（商品を見せながら）
[8〜12秒]  「〇〇が△△になったのがすごくよくて」（使用感・変化を語る）
[12〜15秒] 「@ブランドアカウント チェックしてみて！」（テキスト＋ロゴ）
```

### UGC風広告を撮影する際の注意点

**やること:**

- スマートフォンを手持ちで撮影（わずかなブレがリアル感を出す）
- 自宅や日常的な場所を背景にする
- ナチュラルメイク・普段着で出演する（または商品だけのシーン）
- 字幕（テロップ）をつける（音無しでも伝わるように）

**やってはいけないこと:**

- 虚偽の体験談を語る（「〇日で〇kg痩せた」などの根拠のない効果主張）
- 他人の動画を無断使用する
- 「PR」「広告」の表示を隠す（Metaの規約違反かつ景品表示法上の問題）

## 5-8: クリエイティブのA/Bテスト設計

1つのクリエイティブで「これで完璧」と判断するのは早計です。複数のバリエーションを試して、最も効果的なものを見つけていきましょう。

### テストすべきクリエイティブ要素

| テスト要素 | 変数例A | 変数例B |
|-----------|---------|---------|
| 冒頭の訴求 | ブランド名を最初に出す | 問題提起から入る |
| フォーマット | 静止画 | リール動画 |
| クリエイティブスタイル | プロ撮影風 | UGC風 |
| CTAボタン | 「詳細はこちら」 | 「プロフィールを見る」 |
| テキスト長 | 短文（80字以内） | 中文（150字以内） |

### A/Bテストの実施手順

1. 広告セットを2つ作成します（設定はターゲット含め同じ）
2. 各広告セットに異なるクリエイティブを1つずつ設定します
3. 同じ予算を配分します（例: 各500円/日）
4. 2週間以上運用してデータを蓄積します
5. CPM・リーチ数・広告想起リフト（推定値）を比較します

> **重要**: テストは一度に1つの要素だけ変えてください。複数の要素を同時に変えると、どの変化が効果に影響したか判断できなくなります。

## 5-9: 認知広告クリエイティブのチェックリスト

制作・入稿前に以下を必ず確認しましょう。

**ビジュアル面:**

- [ ] 縦型（9:16）のサイズになっているか（リール・ストーリーズの場合）
- [ ] ロゴが左上に配置・常時表示されているか
- [ ] セーフゾーン（上下150px）に重要な情報が入っていないか
- [ ] 動画の冒頭3秒以内にブランド名・ロゴが表示されているか
- [ ] 文字が小さすぎてスマホ画面で読めないサイズになっていないか

**テキスト面:**

- [ ] 見出し（ヘッドライン）にブランド名が入っているか
- [ ] プライマリテキストが80〜150文字以内か
- [ ] 絵文字の使用はほどほどか（使いすぎると審査で引っかかることも）
- [ ] 誇大表現・根拠のない効果主張がないか

**テクニカル面:**

- [ ] 動画ファイルはMP4形式か
- [ ] ファイルサイズが4GB以下か（Meta広告の制限）
- [ ] 音声が入っているか（音なし視聴用に字幕も用意できるとなお良い）
- [ ] フレームレートが30fps以上か

## 第5章まとめ

認知広告のクリエイティブで覚えておくべき最重要法則を整理します。

**5つの鉄則:**

1. **冒頭3秒でブランドを表示**: 広告想起率を131%高める最重要施策
2. **ロゴは左上に固定配置**: 動画全体を通じて常時表示
3. **見出しにブランド名を入れる**: CTRを28%高める
4. **製品デモを見せる**: 抽象的なメッセージより1.5倍の認知効果
5. **UGC風で自然に届ける**: 広告感を薄め、スクロールを止める

**フォーマット選択の目安:**

| 目標 | 推奨フォーマット | 理由 |
|------|---------------|------|
| 多くの人にリーチ | リール | CPMが最安値 |
| クリックを促す | ストーリーズ | CTRが最高値 |
| じっくり見せる | フィード | 視認時間が長い |

ここまでの3章（第3〜5章）で、認知キャンペーンの設定・ターゲット設計・クリエイティブ制作の基礎が揃いました。次の章では、広告を出稿した後の「効果測定と改善サイクル」について解説します。数字の読み方と、予算を増やすタイミングの判断基準を身につけていきましょう。
