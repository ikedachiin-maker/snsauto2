# 第11章: 【業種別】実践ケーススタディ5選

---

理論を学んだ後、最も知りたいのは「実際にどうやって動かすのか」という具体的なイメージではないでしょうか。

この章では、5つの業種について、それぞれの事業規模・予算・目標に合わせたInstagram認知広告の実践事例をケーススタディ形式で紹介します。いずれも架空の事業者を想定した仮想シナリオですが、各数値は業界標準・実績データをもとにしたリアルな想定値です。自分のビジネスに近い事例を参考に、すぐに実践できる部分から取り組んでみてください。

---

## ケース1: 美容サロン（月1万円・地域限定配信）

### 事業概要

仮に「カラーリゾート・Hana」という、東京都世田谷区にある個人経営のヘアサロンがあるとします。オーナー兼スタイリストの1人営業で、席数は3席。口コミと紹介で顧客を獲得してきたが、新規集客が課題になっていました。

月の広告予算は1万円。費用を抑えながら、認知を広げて新規予約を獲得したい、というのがスタート地点です。

### ターゲット設定

| 項目 | 設定内容 |
|------|---------|
| 地域 | 世田谷区・目黒区・渋谷区（店舗から半径5km圏内） |
| 年齢 | 25〜44歳 |
| 性別 | 女性 |
| 興味 | 美容・ヘアケア・ファッション |
| プラットフォーム | Instagramのみ（フィード＋ストーリーズ） |

地域限定配信のメリットは、「会いに来られる距離の人」だけに予算を集中できることです。全国配信と同じ予算でも、ターゲットエリアを絞ることでインプレッション単価が下がり、より多くのターゲットにリーチできます。

### クリエイティブ例

**広告形式**: 正方形画像（1:1）× 2パターンでA/Bテスト

**パターンA（ビフォー・アフター）**:
施術前と施術後を左右に並べたカラー剤の仕上がり写真。テキストオーバーレイに「世田谷×ダメージレスカラー」を表示。キャプションに「傷まないカラーを探している方へ。初回限定20%オフ。プロフィールのリンクから予約受付中」と記載。CTAボタンは「詳しくはこちら」。

**パターンB（スタイリスト顔出し）**:
オーナーが笑顔でカメラを見ている写真。テキストオーバーレイは「カラー専門9年のプロが担当」。個人サロンにおいて「誰が担当するか」は大きな信頼材料になるため、顔出し広告は高いクリック率を出しやすいです。

### KPI設定

| KPI | 目標値 |
|-----|-------|
| リーチ数（月間） | 2,000〜4,000人 |
| プロフィール訪問数 | 200人以上 |
| 新規フォロワー獲得 | 20〜30人/月 |
| 広告経由予約 | 3〜5件/月 |

### 結果の数値例（3ヶ月後）

- 月間リーチ数: 3,200人（予算1万円 / CPM約3,100円）
- プロフィール訪問数: 310人/月
- 新規フォロワー: 月平均28人
- 広告経由の新規予約: 月4〜5件
- 平均客単価1万円と仮定すると、月4万〜5万円の売上増加
- 広告費1万円に対してROI約400〜500%を達成

**成功の要因**: 地域を絞ったことで無駄なリーチをなくし、限られた予算でも十分な頻度でターゲットに広告を表示できた。また、オーガニックでは毎週施術写真を3〜4枚投稿し、プロフィール訪問者がフォローしたくなるアカウントを整備した。

## ケース2: 飲食店（月3万円・Reels中心）

### 事業概要

仮に「ビストロ・Fuji」という、神奈川県横浜市に位置するフレンチビストロがあるとします。客席20席のこぢんまりとしたお店で、ランチとディナーを営業中。SNSは以前からやっていたものの、投稿内容が料理写真ばかりで反応が伸び悩んでいました。

月3万円の広告予算で、週末のディナー予約を増やすことが目標です。

### ターゲット設定

| 項目 | 設定内容 |
|------|---------|
| 地域 | 横浜市内全域 + 隣接する川崎市・鎌倉市 |
| 年齢 | 28〜50歳 |
| 性別 | 指定なし（カップル・ファミリー層） |
| 興味 | グルメ・レストラン・ワイン・記念日 |
| 詳細ターゲット | 「記念日」「誕生日」の行動ターゲティング |
| プラットフォーム | Instagram（Reels中心） |

「記念日」や「誕生日」の行動ターゲティングは、飲食店にとって非常に強力なオプションです。近く誕生日や記念日を迎えるユーザーはレストランを探している可能性が高く、広告との関連性が高まります。

### クリエイティブ例

**広告形式**: Reels動画（縦型9:16、30秒）

**動画構成**:
- 0〜3秒（フック）: シェフが鍋で炎を上げているダイナミックな厨房映像。テキストオーバーレイ「横浜で一番のアニバーサリーディナーをあなたに」
- 4〜15秒（本題）: コースの各料理が皿に盛り付けられていく様子。BGMは落ち着いたジャズ。テキストで各料理名を表示
- 16〜25秒（信頼形成）: お客様の笑顔のシーン（許可取得済み）と「Googleレビュー4.7星・口コミ数230件」のテキスト
- 26〜30秒（CTA）: 「記念日・誕生日のご予約はプロフィールリンクから」テキストと予約サイトURL

月3万円の予算のうち、約2万円をReels広告に、残り1万円をフィード広告に配分します。

### KPI設定

| KPI | 目標値 |
|-----|-------|
| Reels再生数（月間） | 5,000回以上 |
| プロフィール訪問数 | 400人以上 |
| 予約サイトへのリンクタップ | 50回以上 |
| 広告経由ディナー予約 | 10〜15組/月 |

### 結果の数値例（3ヶ月後）

- Reels月間再生数: 8,400回（オーガニック含む）
- プロフィール訪問数: 520人/月
- 予約リンクタップ: 68回/月
- 広告経由ディナー予約: 月平均13組
- ディナー客単価2人で8,000円と仮定すると、月104,000円の売上増加
- 広告費3万円に対してROI約246%

**成功の要因**: 「記念日」という感情的な動機に訴えるコピーとReelsの組み合わせが高いエンゲージメントを生んだ。また、週1本のReels投稿（店内の雰囲気・調理風景・スタッフ紹介）がオーガニックでの認知拡大にも貢献した。

## ケース3: ECショップ（月5万円・Advantage+活用）

### 事業概要

仮に「NaturalSkin Store」という、ナチュラル系スキンケア商品を販売するECショップがあるとします。自社のECサイトとInstagramショッピング機能を連携済み。主力商品は5,800円のオーガニック洗顔フォームで、リピート率は高いが新規顧客獲得が課題でした。

月5万円の予算で、Meta Advantage+（旧Advantage+ ショッピングキャンペーン）を活用した新規獲得に取り組みます。Advantage+とは、Meta AIがターゲティング・配置・クリエイティブの最適化を自動で行う機能です（詳細は第6章参照）。

### ターゲット設定

Advantage+を使用する場合、ターゲティングはMetaのAIに大きく委ねる形になります。初期設定として以下のシグナルを提供します。

| 項目 | 設定内容 |
|------|---------|
| コンバージョン目標 | ウェブサイト購入（Metaピクセル設置済み） |
| キャンペーン種別 | Advantage+ ショッピングキャンペーン |
| 既存顧客データ | カスタムオーディエンス（過去購入者のメールリスト） |
| 除外設定 | 過去90日間の購入者を除外 |
| 配信先 | Meta全配置（Advantage+配置） |
| 日予算 | 約1,650円（月5万円） |

### クリエイティブ例

Advantage+キャンペーンでは、複数のクリエイティブをアップロードすることでAIが自動的に最適な組み合わせをテストします。以下の5パターンを用意しました。

1. **商品単体写真（白背景）**: 主力商品の正面アップ。清潔感・高品質感を訴求
2. **ライフスタイル写真**: 朝の洗面台で商品を使う女性の手元写真。「自然派ライフスタイル」の世界観
3. **成分説明インフォグラフィック**: 「石油系界面活性剤ゼロ・パラベンフリー・97%自然由来成分」をビジュアル化
4. **UGC（ユーザー投稿）風動画**: 実際のお客様にお願いして撮影してもらった「使ってみた」動画（許可取得済み）
5. **ビフォー・アフター写真**: 「使い始めて2週間後の変化」（お客様提供、許可取得済み）

CTAはすべて「今すぐ購入」。Instagramショッピング機能と連携しているため、広告タップ→商品ページ→購入が2タップで完結します（CVR+40%の効果）。

### KPI設定

| KPI | 目標値 |
|-----|-------|
| リーチ数（月間） | 15,000〜25,000人 |
| 広告クリック率（CTR） | 1.5%以上 |
| 購入単価（CPA） | 3,000円以下 |
| 月間広告経由購入数 | 15件以上 |
| 広告費用対効果（ROAS） | 1.5倍以上 |

### 結果の数値例（3ヶ月後）

- 月間リーチ数: 21,000人
- 広告クリック率: 1.8%
- 月間クリック数: 378回
- 購入数: 月平均19件
- CPA（顧客獲得単価）: 2,630円（目標3,000円を下回る）
- 売上: 19件 × 5,800円 = 110,200円
- ROAS: 110,200円 ÷ 50,000円 = 約2.2倍

**成功の要因**: Advantage+にUGC動画をクリエイティブとして提供したことで、広告らしくない自然な表示がCTRを押し上げた。また、Instagramショッピングとの連携により購入までの導線が短縮され、CVRが大きく改善した。2ヶ月目以降はAIの学習が進み、CPAが当初の3,500円から2,600円台まで改善された。

## ケース4: コーチング/スクール（月2万円・リードマグネット連動）

### 事業概要

仮に「ビジネスコーチ・田中昇さん」が運営する、会社員向けの副業スタートコーチングがあるとします。個別コーチング（3ヶ月プログラム・15万円）を販売していますが、「高額サービスをいきなり広告で売る」のは難しいため、リードマグネット（無料の特典）を入口にした2段階の導線を設計しました。

リードマグネットとは、メールアドレスやLINE登録と引き換えに無料で提供するコンテンツのことです（詳細は第7章参照）。

### ターゲット設定

| 項目 | 設定内容 |
|------|---------|
| 地域 | 日本全国 |
| 年齢 | 25〜40歳 |
| 性別 | 指定なし |
| 興味 | 副業・フリーランス・キャリアアップ・自己啓発 |
| 行動 | ビジネス書購読者・オンライン学習ユーザー |
| プラットフォーム | Instagram（フィード＋ストーリーズ） |

### クリエイティブ例

**広告の目的**: 無料PDF「会社員のまま月5万円稼ぐ副業ロードマップ」のダウンロード（LINE友だち登録）

**広告コピー（キャプション）**:
```
会社員のあなたへ。

「副業を始めたいけど、何から手をつければいいかわからない」

そんな悩みを持つ方のために、副業スタートの全手順を
1枚のロードマップにまとめました。

✅ 副業の種類と選び方
✅ 最初の案件の獲得方法
✅ 本業との両立スケジュール管理術

今すぐダウンロード（無料）▼
↓プロフィールのリンクからLINEに登録するだけ
```

**広告画像**: 田中さん本人が写っている写真（顔出し）+「無料PDF配布中」のテキストオーバーレイ。コーチング系は「人物の信頼感」が最も重要なため、顔出し広告が高い効果を発揮します。

### 2段階の導線設計

```
[広告を見る]
     ↓
[LINE友だち登録 + 無料PDF受け取り]
     ↓
[LINEで3日間の価値提供コンテンツを自動配信]
     ↓
[個別無料相談の案内]
     ↓
[有料コーチング（15万円）の申し込み]
```

月2万円の広告費は、LINE登録という低単価のコンバージョンを目標にしているため、比較的少ない予算でも十分なリーチが可能です。

### KPI設定

| KPI | 目標値 |
|-----|-------|
| LINE登録数（月間） | 50〜100件 |
| LINE登録単価（CPL） | 300円以下 |
| 無料相談申し込み率 | LINE登録者の10% |
| 有料成約率 | 無料相談者の20% |
| 月間有料成約数 | 1〜2件 |

### 結果の数値例（3ヶ月後）

- 月間LINE登録数: 78件
- LINE登録単価: 256円（目標300円以下を達成）
- 無料相談申し込み: 78件 × 10% = 約8件/月
- 有料成約: 8件 × 20% = 約1.6件/月（月によって1〜2件）
- 売上（平均1.5件 × 15万円）: 225,000円/月
- 広告費2万円に対してROI約11倍

**成功の要因**: 高額商品をいきなり販売しようとせず、「無料PDF」という低ハードルの入口を設けたことで、まず接点（LINE登録）を大量に獲得できた。LINE自動配信でのコンテンツ提供が信頼構築を自動化し、田中さんが対応するのは「無料相談」以降に絞られた。また、オーガニック投稿では毎週「副業ノウハウTips」を発信し続けたことで、広告経由でフォローした人たちとの継続的な関係を維持できた。

## ケース5: 地域店舗（月1万円・来店促進）

### 事業概要

仮に「文具・雑貨店 カミノムスメ」という、埼玉県大宮駅から徒歩5分にある小さなセレクト文具・雑貨店があるとします。オーナーが1人で運営しており、地域の固定客はいるものの、新規来店客の獲得が長年の課題でした。ECサイトは持っておらず、あくまで「実店舗への来店」を増やすことが目標です。

月1万円という最小予算で、どこまで来店数を伸ばせるかに挑戦します。

### ターゲット設定

| 項目 | 設定内容 |
|------|---------|
| 地域 | 大宮駅から半径3km圏内 |
| 年齢 | 20〜45歳 |
| 性別 | 女性（主要顧客層） |
| 興味 | 文房具・手帳・インテリア・クラフト |
| カスタムオーディエンス | 既存フォロワーに類似したLookalike（類似オーディエンス） |
| プラットフォーム | Instagramストーリーズ＋フィード |

来店促進の場合、「Map（地図）」のCTAを使ってGoogleマップやAppleマップに直接誘導することが可能です。また、Instagramの「ストア訪問」目標キャンペーンを設定することで、実店舗への来店コンバージョンをMetaが最適化してくれます。

### クリエイティブ例

月1万円の予算で最大のインパクトを出すため、クリエイティブは「季節感×限定感」を活用しています。

**例（10月の場合）**:

ストーリーズ広告（縦型15秒動画）として、秋の限定入荷アイテムを15秒で紹介する動画を制作します。

- 0〜3秒: 秋らしいモーブカラーの文具アイテムを並べたフラットレイ写真。テキスト「今週入荷しました」
- 4〜10秒: 各アイテムを手に取りながら紹介。BGMは秋らしいアコースティックギター
- 11〜15秒: 店舗外観の写真＋「大宮駅から徒歩5分 / 火〜日曜営業 / 地図はこちら」のCTA

**季節のイベントに合わせた展開例**:
- 9〜10月: 秋の新学期・手帳の季節（来年の手帳入荷情報）
- 11〜12月: クリスマスプレゼント向けラッピング無料サービス
- 1〜2月: バレンタインのメッセージカード特集
- 3〜4月: 新生活スタートの文具セットアップ

季節ごとにクリエイティブを更新することで、広告の鮮度を保ちながら関連性スコアを維持します。

### KPI設定

| KPI | 目標値 |
|-----|-------|
| 広告リーチ数（月間） | 1,500〜3,000人 |
| 地図タップ数（月間） | 30回以上 |
| プロフィール訪問数 | 100人以上 |
| 新規来店客（月間） | 10〜15人 |

来店数の測定方法: 「インスタを見てきました」と言ってもらった方に特典（スタンプカードにスタンプ追加など）を提供することで、広告経由の来店数を把握します。

### 結果の数値例（3ヶ月後）

- 月間リーチ数: 2,400人
- 地図タップ数: 42回/月
- プロフィール訪問数: 165人/月
- 新規フォロワー: 月平均22人
- 「インスタを見てきました」来店数: 月12〜18人（平均15人）
- 新規来店客の平均客単価: 2,400円
- 広告経由売上: 15人 × 2,400円 = 36,000円/月
- 広告費1万円に対してROI約260%

さらに、フォロワーが月22人増えていることで、翌月以降のオーガニックリーチが拡大。3ヶ月でフォロワーが66人増加し、毎月の広告なしでの新規接触数も増えていく好循環が生まれました。

**成功の要因**: 地域を極限まで絞ったことで、1万円という最小予算でも同じ人に複数回広告が表示され「あのお店、気になるな」と記憶に残る効果（フリークエンシー効果）が生まれた。また、季節感のあるクリエイティブが「今行く理由」を生み出し、来店への背中を押す役割を果たした。

## 5つのケーススタディから見えてくる共通法則

5つの事例を振り返ると、業種や予算が異なっても、成功に共通するポイントが見えてきます。

**共通法則1: ターゲットを絞るほど効果が上がる**

地域・年齢・興味を広く設定するほど1リーチあたりのコストが上がり、関連性の低い人に広告が表示されます。「自分のお客様はどんな人か」を具体的にイメージして設定することが、すべての基本です。

**共通法則2: 広告の目的は「今すぐ売る」ではなく「接点を作る」**

ケース4のコーチングのように高額商品であれば「LINE登録」、ケース5の地域店舗であれば「地図タップ・フォロー」という、低ハードルの最初のアクションを目標に設定することが重要です。

**共通法則3: オーガニック投稿との連動が効果を倍増させる**

どのケースでも、週複数回のオーガニック投稿（特にReels）が広告効果を補完する役割を担っています。広告で接触→プロフィール訪問→オーガニック投稿で信頼→アクションという流れを設計することが、少ない予算で最大の効果を生む鍵です。

**共通法則4: 数値を計測して改善を繰り返す**

成功した事例はいずれも、最初から完璧なクリエイティブだったわけではありません。A/Bテストや週次の数値確認によって「効果のあるもの」を見極め、予算を集中させた結果です。

## まとめ

5つのケーススタディを通じて、月1万円から5万円という現実的な予算でも、Instagram認知広告が十分なROIを生み出せることがわかりました。

重要なのは「完璧な広告を最初から作ること」ではなく、「小さく始めて、データを見て、改善し続けること」です。

次章（第12章）では、これらの実践を継続するための広告管理ルーティンと、よくある失敗パターンとその対処法を解説します。
